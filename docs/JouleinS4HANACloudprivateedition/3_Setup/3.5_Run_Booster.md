1. If necessary, navigate to your BTP Global Account.<br/>
![run_booster](./Images/NavGloAcc.png)

2. From the Navigation Pane on the left, click **Boosters** >> search for **Joule** >> click **Start** on Setting up Joule tile.<br/>
![run_booster](./Images/StartBooster.png)

3. Confirm the pre-requisites check is successful and click **Next**.</br>
![run_booster](./Images/PreCheck.png)

4. From the Subaccount dropdown select the **Joule** subaccount created earlier and click **Next**.</br>
![run_booster](./Images/SelectSubpng.png)

5. From the Products dropdown for SAP **S/4HANA Cloud, private edition**, you do not need to select anything. However, you can add **SAP Build Work Zone, standard edition** if you are planning to integrate Joule with **SAP Build Work Zone, standard edition (Plan â€“ Standard)**</br>
 Choose whether the integration is for **Testing or Production** and click **Next**.</br>
![run_booster](./Images/SelectInte.png)

6. From the Capability Pacakages dropdown select **S/4HANA Cloud Private Edition**. Click **Next**.<br>
![run_booster](./Images/Capability.png)

7. Leave the default **Formation Name**.  Under the Include System area choose the following and click **Next**. </br>
  SAP Build WorkZone System: **Choose the SAP Build Work Zone created earlier**</br>
  SAP Cloud Identity Services System: **Choose the Cloud Identity Services Tenant**</br>
  S/4HANA Cloud Private Edition System: **Choose the S/4HANA Cloud Private Edition**</br>
![run_booster](./Images/Formation.png)

10. Review the configuration settings and click **Finish**.</br>
  ![run_booster](./Images/FinishBooster.png)

11. Confirm the booster execution is successful and click **Close**.</br>
![run_booster](./Images/SuccessBooster.png)
